  • 复制粘贴选中字符串: 命令行模式下在按下v就会从光标处开始选择,通过移动光标选中要选择的字符，按下y表示复制，按下d表示剪切，最后在要粘贴的地方按下p即可。

    • d$剪切从当前位置到行尾，d^剪切从当前位置到行首。

vim分屏

在写代码的时候分屏可以带给我们很大便利，比如C++代码中时常进行类的实现时会忘记定义中声明的函数，可以开两个屏以便于我们进行code。

分屏相关
  • 在命令行模式下可以通过 :new 进行分屏，new后可以跟一个文件名，在新分出的屏中即可实现打开或新建相应的文件。快捷键ctrl+w，然后按下n键。
  • :split 表示水平分屏，分为上下两个屏，后可跟文件名。
  • :vsplit 表示垂直分屏，分为左右两个屏，后可跟文件名。
  • :only 表示取消分屏，取消当前光标所在的屏。
  • ctrl+w,c关闭当前屏。和上一个:only结果一样。
  • ctrl+w,o也是关闭。
载入文件
  • :vs 文件路径/文件名 在新的垂直分屏中打开文件
  • :sv 文件路径/文件名 在新的水平分屏中打开文件
切换各屏快捷键
  • Ctrl + w, w 切换到后一个屏。
  • Ctrl + w, p 切换到前一个屏。
  • 四个方向切换屏,其中h，i，k，l可以替换为各相应的方向键。
    • Ctrl + w, h
    • Ctrl + w, i
    • Ctrl + w, k
    • Ctrl + w, l
